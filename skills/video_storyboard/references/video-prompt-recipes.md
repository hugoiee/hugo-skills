# Seedance 2.0 视频 prompt 配方

SKILL.md Step 4 调用本文件。本文件只讲"如何把 Step 3 已经出好的 `shot-NN.png` 在 Seedance 2.0（火山方舟）里跑成 5 秒视频"。

---

## Seedance 2.0 模型特点

- **中文友好** — 中文 prompt 直接用，文化意象理解准
- **角色一致性强** — 支持 `@图片名` 引用素材；首帧+文模式下一致性最稳
- **物理理解到位** — 风、水、布料、光影动力学自然
- **运镜指令精准** — 中文运镜术语理解清晰
- **4–15 秒单段** — 默认 5s、24fps；长镜头要分段拼接

**短板与规避：**

| 短板 | 规避策略 |
|------|---------|
| 高度写实人脸有概率被审核拦截 | 用风格化处理 / 背影侧脸 / 剪影 |
| 默认 16:9，宽银幕需后期裁切 | prompt 提示"主体居中"，或 `ratio=21:9` |
| 音频生成不达电影级 | 抽卡阶段全程 `generate_audio=false`，后期 Suno + 音效库 |

## 入口选择

| 入口 | 适用 | 一致性 | 难度 |
|------|------|--------|------|
| **首帧+文**（推荐默认） | 已用 ChatGPT-Images 2 出好图 | ⭐⭐⭐⭐⭐ | 低 |
| 文生视频 | 找不到合适首帧 / 抽象镜头 | ⭐⭐ | 高 |
| 首尾帧 | 需要明确动作起止 | ⭐⭐⭐⭐ | 中 |
| 全能参考（多图+文） | 多人 / 多道具一致性 | ⭐⭐⭐ | 中 |
| 视频延长 | 长镜头拼接 | ⭐⭐⭐⭐ | 中 |

**本 skill 默认走"首帧+文"**，把 Step 3 的 `shot-NN.png` 当首帧。

---

## 视频 prompt 黄金结构

```
首帧：shot-NN.png（在火山方舟界面手动上传）

文字 prompt：
[主体动作（1–2 句短句，谁做什么）]
+ [镜头运动术语（单一或简单复合，速度副词必加）]
+ [氛围一句话（光、风、粒子、声音感）]

Negative: [全局负面词]
```

**写作要点：**
- 单镜头只用 1–2 个运镜，超过 3 个动作会乱
- 速度词必须明确："缓慢" / "中速" / "快速"
- 与节奏定位对应：节奏"静"用固定机位，节奏"崩"用"快速推进+手持晃动"
- 不要重复描述首帧已有的内容（角色长相、场景细节），首帧图已经表达了

## API 参数完整速查（2026-04 火山方舟开放）

| 参数 | 取值 | 默认 | 备注 |
|------|------|------|------|
| `model` | `doubao-seedance-2-0-260128`（标准）/ `-fast-260128`（Fast） | — | Fast 版省钱、速度快、质量略低，**抽卡阶段先用 Fast** |
| `content` | 多模态数组：`text` / `image_url` / `video_url` / `audio_url` | — | 上限：图 9 张 (30MB)、视频 3 段 (50MB)、音频 3 段 (15MB) |
| `resolution` | `480p` / `720p` / `1080p` / `2K` | `1080p` | 抽卡 720p 省 token，定稿 1080p 起 |
| `ratio` | `16:9` / `9:16` / `4:3` / `3:4` / `21:9` / `1:1` / `adaptive` | `16:9` | 宽银幕用 `21:9` 或后期裁切 |
| `duration` | 4–15（秒） | `5` | 长镜头分段，每段衔接靠 `return_last_frame` |
| `framepersecond` | 整数 | `24` | 默认就够 |
| `watermark` | bool | — | 平台水印开关 |
| `generate_audio` | bool | `false` | **抽卡阶段必须 false** |
| `return_last_frame` | bool | `false` | 长镜头拼接靠它返回末帧 |
| `camerafixed` | bool | `false` | 强制锁定机位，**比 prompt 写"静止"更稳** |
| `seed` | int | 随机 | 好镜头务必记录用于复现 |
| `callback_url` | URL | — | 任务完成回调，免轮询 |

| 类别 | 运镜 | Prompt 写法 | 适用 |
|------|------|------------|------|
| 固定 | 固定机位 | "镜头完全静止" / "固定机位" + 开 `camerafixed=true` | 留白、特写、氛围镜 |
| 推拉 | 推镜头 | "镜头缓慢推近" / "快速推进" | 强调主体、情绪聚焦 |
| 推拉 | 拉镜头 | "镜头缓慢拉远" | 揭示环境、落幕感 |
| 横移 | 摇镜头（水平） | "镜头从左向右摇" / "横摇" | 展示空间 |
| 横移 | 摇镜头（垂直） | "镜头向上摇" / "俯仰摇" | 揭示高度、仰望震撼 |
| 平移 | 平移镜头 | "镜头水平平移" / "侧向移动" | 追随主体、横向叙事 |
| 跟随 | 跟拍 | "镜头跟随主体" / "跟拍" | 角色行进、动作戏 |
| 跟随 | 环绕 | "镜头围绕主体环绕一周" | 强调主体存在感 |
| 升降 | 升镜头 | "镜头上升" | 揭示规模、空灵感 |
| 升降 | 降镜头 | "镜头下降" | 沉重感、压迫感 |
| 复合 | 推+摇 | "镜头一边推近一边向左摇" | 复杂情绪转场 |
| 复合 | 移+跟 | "侧向平移跟拍主体" | 公路片、追逐戏 |
| 视角 | 第一人称 | "POV 主观镜头" | 沉浸感 |
| 视角 | 俯/仰视 | "俯拍视角" / "仰拍视角" | 权力关系、心理张力 |
| 特殊 | 手持感 | "轻微手持晃动" | 纪实感、紧张感 |
| 特殊 | 希区柯克变焦 | "镜头推近的同时变焦拉远（dolly zoom）" | 心理冲击瞬间 |

## 5 类节奏的 prompt 配方对应表

| 节奏定位 | 镜头运动 | 速度词 | camerafixed | 示例 prompt |
|---------|---------|--------|-------------|------------|
| 静 | 固定 | — | `true` | "镜头完全静止，主角的目光缓缓抬起" |

## 抽卡难度评级标准

| 难度 | 镜头类型 | 预估次数 |
|------|---------|---------|
| ⭐⭐ | 远景 / 剪影 / 道具空镜 / 固定机位 | 3–5 次 |
| ⭐⭐⭐ | 中景 / 简单运镜 / 单人动作 | 5–8 次 |
| ⭐⭐⭐⭐ | 多人互动 / 复合运镜 / 群体一致性 | 8–15 次 |
| ⭐⭐⭐⭐⭐ | 面部特写 + 表演 / 多生物互动 / 复杂物理 | 10–20 次 |

## 抽卡顺序（先易后难，建立信心和手感）

```
第 1 批  ⭐⭐    远景 / 剪影 / 空镜     # 先跑通流程，建立信心
第 2 批  ⭐⭐⭐⭐⭐ 人脸一致性硬骨头     # 趁手感好啃硬骨头
第 3 批  ⭐⭐⭐   中等难度环境镜        # 常规批量
第 4 批  ⭐⭐⭐⭐  动作戏 / 视觉重点    # 进入高密度
第 5 批  ⭐⭐⭐⭐⭐ 开场镜头           # 放最后做，工具脾气已熟
```

## 后期工作流（抽卡之后）

**⚠️ 抽卡阶段全程 `generate_audio=false`。** 理由：AI 音频质量达不到电影级、音频绑定视频无法独立调整、浪费 token。

```
1. 视频抽卡（无音频）
2. 剪辑成片（无声版，DaVinci Resolve / Premiere）
3. Suno AI 生成主音乐
4. 按音乐节奏微调剪辑切点
5. freesound / Splice / Epidemic Sound 加音效
6. DaVinci Resolve Fairlight 混音
7. DaVinci Resolve Color 调色（按视觉铁律的色板做一次统一调色 LUT）
8. 字幕：思源宋体 / 苹方黑 / Sweet Sans（按风格选）
```

## 复用建议

每个项目跑完，把以下三件事存进项目目录的 `_seeds.md`：

1. **出好的镜头的完整 prompt + seed + 参数**（下次类似镜头直接复用）
2. **本项目踩过的坑 + 规避方法**（项目复盘）
3. **角色档案图 + 视觉铁律**（下次同 IP 续作直接用）
