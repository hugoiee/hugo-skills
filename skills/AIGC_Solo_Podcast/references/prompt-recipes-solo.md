# 单人口播 图生视频 prompt 配方（即梦 / 可灵 / Vidu）

SKILL.md **Step 2 / Step 3 / Step 5** 调用本文件。只讲**首帧怎么出、逐段说话 prompt 怎么写、工具怎么选、怎么验收抽卡、数字段怎么配图**。切段 / 估时 / 防发呆 → `segment-and-pacing.md`。

---

## ⚠️ 两条铁律先记住

**铁律一：说话片段必须开原生音频。** 口型和声音由工具一次出（音画一体），关了音频抽出来口型对不上后配音、整段废。用带原生音频的版本：**可灵 3.0 / 即梦 Seedance 2.0 / Vidu Q3**（⚠️ Vidu Q2 不带，要升 Q3）。

**铁律二：prompt 正文必须干净、可直接复制。** 代码块里**只放纯 prompt**——能一键粘贴进即梦 / 可灵 / Vidu 文本框就出片。`首帧文件、音频开关、时长、情绪标签`是元数据，写在代码块**外**的描述行，绝不混进 prompt。

- ❌ prompt 里出现 `首帧：host_solo.png` / `[开音频，duration=13s]` / `情绪：吐槽`。
- ✅ prompt 只描述画面：host 形象 + 固定机位 + 逐字台词 + 情绪外化身体细节 + 连贯讲话约束。

---

## (1) 单人首帧 `host_solo.png` 配方

全集只出这一张，所有段都用它当首帧。

```
[host 稳定特征词，例：35–40岁亚洲男性、方圆脸、神情精明、深色西装内搭浅色衬衫、利落短发]，端坐在简洁演播室里，身后虚化书架 / 纯色背景，暖冷适中灯光，背景静止。中近景 medium close-up、eye-level 平视、固定机位、面向镜头，浅景深背景虚化。
Negative：运镜、推拉摇移、变形脸、多余的人、文字水印、卡通风格。
```

⚠️ 锁机位词必带：`固定机位 fixed / eye-level / medium close-up 中近景 / 背景不变 background unchanged`。全集复用同一张，角色 / 机位 / 背景 / 光照不漂。

---

## (2) 逐段说话 prompt 卡（交付形态）

每段一张卡，**代码块只放纯 prompt，元数据在外**：

- **段头**：`段号 ｜ 片段ID ｜ ~时长 ｜ 情绪`
- **台词**：逐字（⚠️ 台词内部若有引号，一律换「」或书名号，避免与外层台词引号同形、复制时断错）
- **描述行**：`首帧 host_solo.png ｜ 开原生音频 ｜ ~Xs（按台词实际长度设时长）`
- **代码块**（纯 prompt），结构：

```
[host 形象一句] + 中近景固定机位 fixed、禁止任何运镜 + 对着镜头连贯说出这句台词："<逐字台词>" + [情绪外化身体细节] + 全程连贯讲话、话音不断、不发呆不留空档 + 口型与声音精确逐字同步、机位全程不动、背景静止。
```

### 范例（干净，可直接复制）

段头：`段 03 ｜ host_say_03 ｜ ~11s ｜ 嘲讽 → 激动`
台词：“4900相对目前4000左右的价格，依然有20%的空间啊！曾经的「大多头」如此背刺自己，遭罪的就是投资者！！”
描述行：`首帧 host_solo.png ｜ 开原生音频 ｜ ~11s`

```
35到40岁、神情精明带点犀利劲的财经评论男主播，深色西装内搭浅色衬衫，简洁演播室背景虚化书架、暖冷适中灯光，背景静止。中近景medium close-up、eye-level平视、固定机位fixed、禁止任何运镜。对着镜头连贯说出这句台词："4900相对目前4000左右的价格，依然有20%的空间啊！曾经的「大多头」如此背刺自己，遭罪的就是投资者！！"。说数字时手指轻点空中像在算账，说"20%的空间啊"时夸张摊手、挑眉、带嘲讽的笑；说"背刺自己"时身体前倾、眉头紧锁、手往桌面一压，说"遭罪的就是投资者"时提高音量、目光锐利逼视镜头、手指有力地点向前方。全程连贯讲话、话音不断、不发呆不留空档。口型与声音精确逐字同步，机位全程不动，背景静止。
```

## (3) 情绪外化词表（把情绪标签翻成身体细节，别写"严肃"二字）

| 情绪 | 身体外化（写进 prompt） |
|---|---|
| 吐槽 / 讽刺 | 挑眉、嘴角下撇冷笑、轻摇头、抬手虚点像点名、眼神揶揄 |
| 质问 / 反问 | 身体前倾、双手摊开做反问手势、盯紧镜头、语气上挑 |
| 激动 / 强调 | 提高音量、手往桌面一压、手指点向前方、眉头紧锁 |
| 冷静 / 梳理 | 神情沉稳、语速放缓、手掌在身前平推归类、目光平稳 |
| 转折 | 抬手做"且慢"手势、节奏分明逐次加重、说到反转处收起笑意、下巴微收 |
| 语重心长 | 放缓语速、手掌向下按定像放下重物、目光恳切真诚、微微点头 |
| 轻松 / 调侃 | 嘴角带笑、肩膀放松、手势轻快、眼神带戏谑 |

> 写法：`说"<关键词>"时<身体动作>`，把情绪绑到具体台词位置，口型 / 表情更贴台词。

## (4) 防发呆，在 prompt 里怎么写

**权威尾串（每段 prompt 结尾固定带，全 skill 统一以此为准）：** `全程连贯讲话、话音不断、不发呆不留空档`。情绪强的段可再加 `、开口即讲、话音落即收` 加强；本文件范例、SKILL.md、交付物模板均用此核心串。

但 prompt 只是一半——**另一半是生成时长 = 台词实长**（见 `segment-and-pacing.md` 四）。两者缺一不可：时长设太长，再怎么写"不发呆"也会被补帧。

## (5) 工具差异速查（做单人口播视角，截至 2026）

| 维度 | 可灵 3.0 | 即梦 Seedance 2.0 | Vidu Q3 |
|---|---|---|---|
| 原生音画一体 | ✅ | ✅ | ✅（16s，对话 / 旁白音轨）|
| 单段时长上限 | ~15s | 较长，上限随版本（默认 5s ⚠️ 现查）| ~16s |
| 一致性入口 | 主体参考 | `@图片名`、首帧 + 文 | 参考图槽位 |
| 适合 | 口型稳 | 国内顺手、与剪映集成 | 长段（≤16s）、便宜抽卡 |

⚠️ 单段超过你所用工具的时长上限就在 Step 1 拆：可灵约 15s 偏短，要长单段用即梦 Seedance 2.0 或 Vidu Q3（~16s）。版本号 / 时长上限 / 音频开关名随版本变，标 ⚠️ 的用前现查官方。

---

## (6) 每段验收 checklist

- [ ] 口型与声音**逐字同步**，无错位、无飘音
- [ ] **片头片尾不发呆**、全程在讲话（时长贴台词）
- [ ] 机位完全静止、无运镜、背景 / 人物不漂移
- [ ] 面部 / 服装与首帧一致，无崩脸
- [ ] 情绪与台词匹配、外化自然不僵硬
- [ ] 代码块是纯 prompt，无元数据残留

---

## (7) 抽卡省额度 + 数字 / 数据段配图 B-roll

- **低分辨率预览**先验口型 / 构图，定稿再出高清；满意镜头记 seed 复现（⚠️ 各平台 seed 支持不一）。
- 固定机位口播 ⭐⭐⭐（口型是难点），先抽 1–2 条验口型再批量。
- **数字 / 数据段配图**：念"各机构价格 / 涨跌数据 / 排名"的段，做一张**图表 / 要点字幕卡**当 B-roll 盖在口播画面上（**口播原声保留 = 盖画面不盖声音**，与 `AIGC_Podcast_Script` 的 cutaway 同理），比一直拍人耐看。配图可用 MJ / 即梦出图，思路：`极简财经信息图 / 价格区间对比图 / 干净留白、可读大字、品牌色`。⚠️ 盖 B-roll 时口播声音不动，只换上层画面。

---

> 单人口播：一张首帧锁全片，逐段纯 prompt + 情绪外化 + "连贯讲话不发呆"，时长贴台词——复制即抽，片头片尾不傻坐。
