# 叙事结构与时长适配参考

> 用法：阶段 5「分镜骨架搭建」中需要根据用户选择的时长拆解三幕结构时调取此文件。
> 每种时长包含：节奏分配、镜头数量、典型结构、雷区。

---

## 不同时长的三幕分配速查

| 时长 | 第一幕 | 第二幕 | 第三幕 | 镜头数 | 信息密度 |
|------|--------|--------|--------|--------|---------|
| **15s** | 3s | 8s | 4s | 3-4 | 单一信息点 |
| **30s** | 6s | 15s | 9s | 4-6 | 一个核心 + 一个 callback |
| **60s** | 15s | 30s | 15s | 8-10 | 一个核心 + 一条隐藏线 |
| **90s** | 20s | 50s | 20s | 10-13 | 一个核心 + 多条隐藏线 |
| **2 分钟** | 25s | 70s | 25s | 14-18 | 完整故事弧 |
| **3 分钟** | 40s | 100s | 40s | 18-25 | 多线叙事 |

---

## 15 秒：单一信息点

### 适用场景

- 信息流广告
- 短视频平台轮播
- 即时品牌召回
- TikTok / Reels / 视频号

### 三幕结构

```
0-3s    第一幕：钩子（必须前 1.5s 内抓住注意力）
3-11s   第二幕：核心信息（一句话能说完）
11-15s  第三幕：CTA / 落版
```

### 镜头分配（3-4 个）

```
01：钩子镜头（3s）— 一个能让人停下的画面
02：核心展示（5s）— 产品 / 信息主体
03：CTA / 落版（4s）— 行动召唤
+ 可选 04：动作收尾（3s）
```

### 关键规则

- **前 1.5 秒必须出现"违反预期"的元素**（否则用户划走）
- **不要追求完整叙事**，只追求"被记住"
- **每镜必须能独立成图**（用户随时划走，半截画面要能形成印象）

### 雷区

- ❌ 慢起拍（铺垫 3 秒以上）
- ❌ 信息分散（同时说两件事）
- ❌ 落版前画面 = 落版前画面就消失了，CTA 等于没说

---

## 30 秒：一个核心 + 一个 callback

### 适用场景

- 电视广告
- 信息流深度版本
- 产品演示
- 品牌叙事短版

### 三幕结构

```
0-6s    第一幕：建立情境 + 引入角色 / 产品
6-21s   第二幕：核心剧情 / 卖点展示
21-30s  第三幕：情感升华 + CTA + 落版
```

### 镜头分配（4-6 个）

```
01：开场建立（4s）
02：核心镜头 A（6s）
03：核心镜头 B（5s）
04：情感升华（5s）— 与开场形成回响
05：CTA / 落版（5-10s）
```

### 关键规则

- **第二幕中段必须有一个"高潮镜头"**（即使简短）
- **第三幕开始时给一个 callback**（呼应开场）— 让 30s 有"完整感"
- **CTA 不要太长**（5s 已经是上限）

### 节奏曲线

```
3 拍：稳 → 起 → 落
平 → 起 → 平 → 高潮 → 平 → 收
```

### 雷区

- ❌ 第二幕拖沓（最常见崩盘）
- ❌ 没有任何 callback
- ❌ CTA 占了 1/3 时长

---

## 60 秒：一个核心 + 一条隐藏线

### 适用场景

- 标准 TVC
- 品牌故事
- 概念片
- 短预告片

### 三幕结构

```
0-15s   第一幕：建立世界观 + 角色出场
15-45s  第二幕：发展 + 冲突 / 卖点深入
45-60s  第三幕：高潮 + 收尾 + 落版
```

### 镜头分配（8-10 个）

```
第一幕（3 镜）：开场钩子 + 角色出场 + 世界建立
第二幕（4-5 镜）：起 → 承 → 转 → 高潮
第三幕（2-3 镜）：情感收束 + 落版
```

### 关键规则

- **可以塞一条隐藏线**（视觉母题、色彩呼应、动作呼应）
- **必须有 1-2 个 callback 镜头**
- **第二幕的节奏必须有变化**（不能匀速）

### 节奏曲线参考

```
平 → 起 → 起 → 转 → 高潮 → 缓 → 收
```

### 雷区

- ❌ 镜头超过 12 个 → 每镜都没记忆点
- ❌ 第二幕节奏匀速 → 中段疲劳
- ❌ 多于一条隐藏线 → 观众看不到任何一条

---

## 90 秒：一个核心 + 多条隐藏线（最经典宣传片时长）

### 适用场景

- 完整宣传片
- 招商提案视频
- 游戏宣传片
- 个人作品集主作品

### 三幕结构

```
0-20s   第一幕：建立角色 + 世界 + 母题
20-72s  第二幕：核心叙事（可拆为 4-6 个轮回 / 蒙太奇 / 章节）
72-90s  第三幕：高潮 + 收尾 + 落版
```

### 镜头分配（10-13 个）

```
第一幕（3 镜）：母题符号 + 角色锚点 + 世界建立
第二幕（6-8 镜）：发展，必须有节奏起伏（动-静-冷-悠-崩 等）
第三幕（2-3 镜）：高潮回响 + 终章 + 落版
```

### 关键规则

- **必须有完整的隐藏叙事线**（不止视觉母题，要有内涵）
- **第二幕可以拆为多段式**（如本案"五域闪切"）
- **镜头 02 和片尾镜头必须呼应**（建立 callback 闭环）
- **节奏曲线必须复杂**：动-静-冷-悠-崩 这类多段式节奏

### 节奏曲线（本案大爱仙尊用过）

```
慢 → 慢 → 慢 → 动 → 静 → 冷 → 悠 → 崩 → 爆 → 慢 → 慢 → 慢
建立    建立    建立    五域轮回（节奏曲线）        收尾   收尾
```

### 雷区

- ❌ 镜头超过 15 个 → 信息溢出
- ❌ 第二幕用同一类景别（如全是中景） → 单调
- ❌ 没有 callback → 90 秒像一盘散沙
- ❌ 落版前最后一镜不是高潮 → 收不住

---

## 2 分钟：完整故事弧

### 适用场景

- 微电影
- 品牌纪录片短版
- 复杂概念片
- 长版宣传片

### 三幕结构

```
0-25s   第一幕：建立 + 角色塑造
25-95s  第二幕：完整起承转合（可分为 4-5 段）
95-120s 第三幕：高潮 + 收尾
```

### 镜头分配（14-18 个）

第二幕可分为：
- 起（10-15s）
- 承（15-20s）
- 转（15-20s）
- 高潮（15-20s）

### 关键规则

- **必须有完整角色弧光**（变化过程要可见）
- **可以包含 2-3 个明确的"小节"**（用过渡镜头分隔）
- **节奏必须有大起大落**

### 雷区

- ❌ 节奏过于平稳 → 中段流失
- ❌ 角色没有变化 → 像 90s 强行拉长

---

## 3 分钟：多线叙事

### 适用场景

- 短电影
- 复杂概念深度展示
- 多角色品牌叙事

### 结构

可以采用：
- **平行剪辑**（A 故事 + B 故事交替）
- **多章节**（明确分章）
- **嵌套结构**（故事中的故事）

### 雷区

- ❌ 把 90s 的内容硬拉到 3min → 全程拖沓
- ❌ 多线但每条都不深入 → 不如做一条
- ❌ 没有明确章节感 → 观众迷失

**建议：除非有强烈的叙事需求，否则不要做超过 90s 的 AIGC 视频。AI 生成的人物一致性会随时长指数级崩坏。**

---

## 三幕结构的本质

不管时长多少，**三幕的功能永远是**：

| 幕 | 功能 | 用户心理 |
|----|------|---------|
| 第一幕 | 建立 | "这是什么？" |
| 第二幕 | 发展 | "发生了什么？" |
| 第三幕 | 收尾 | "意味着什么？" |

**这三个心理问题如果在对应幕里没回答，整片就崩了。**

---

## 节奏曲线设计

第二幕是节奏曲线的主战场。常用模板：

### 模板 A：三段式（适合 30-60s）

```
起 → 转 → 高潮
快   慢    爆
```

### 模板 B：五段式（适合 60-90s）

```
动 → 静 → 冷 → 悠 → 崩
快   缓   凉   舒   重
```

### 模板 C：起伏式（适合 90s+）

```
起 → 缓 → 起 → 缓 → 高潮 → 缓
```

### 模板 D：渐强式（适合任何时长）

```
最弱 → 弱 → 中 → 强 → 最强
```

## 镜头切点节奏

不同时长的剪辑节奏参考：

| 时长 | 平均镜头时长 | 最长 | 最短 |
|------|-------------|------|------|
| 15s | 4-5s | 7s | 2s |
| 30s | 5-6s | 8s | 2s |
| 60s | 6-8s | 12s | 2s |
| 90s | 6-9s | 12s | 2s |
| 2 分钟 | 7-10s | 15s | 2s |
| 3 分钟 | 8-12s | 20s | 2s |

**AI 生成视频的单镜头上限通常是 5-10 秒**——超过这个时长一致性会崩。如果某镜需要 15 秒，要么拆分，要么用"匹配剪辑"伪装成一镜。

## Callback 设计

让短片"完整"的秘诀是 **callback**——首尾呼应。

### 常用 callback 模式

1. **视觉呼应**：开场和片尾用同一构图（但内容已变）
2. **动作呼应**：开场和片尾用同一动作（但情绪已变）
3. **物件呼应**：开场出现的物件，片尾再次出现
4. **声音呼应**：开场的某个声音元素，片尾再次出现
5. **眼神呼应**：开场镜头是角色看镜头，片尾再来一次

**90s 至少要有 1-2 个 callback，否则会显得散。**

---

## 落版设计

最后 3-10 秒的"落版"是观众离开前的最后印象。

### 落版三种类型

| 类型 | 设计 | 适合 |
|------|------|------|
| **文字落版** | 全黑 + 中央文字 + 静止 | 品牌、IP |
| **画面落版** | 一张定格画面 | 艺术、情感 |
| **CTA 落版** | 行动召唤 + logo | 商业广告 |

### 落版字幕长度

- 标题：4-8 字最佳
- 副标题（可选）：8-16 字
- CTA：6-12 字

**不要在落版塞太多字。**
