## 蒙考拉冷玩

**别称：** Roger Deakins / Denis Villeneuve 系
**代表作：** 《Sicario》《Prisoners》《Blade Runner 2049》《Dune》《Enemy》

### 核心特征

电影摄影师 Roger Deakins 在 Denis Villeneuve 几部片里建立的"冷峻、克制、抽离"的现代电影美学。每一帧都像静物画，但每一帧都暗流涌动。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 单一硬光源 + 大面积阴影，几乎不补光 |
| 色板 | 青灰、墨蓝、铁锈红，禁用饱和色和暖橙 |
| 构图 | 大量负空间，主体压在画面边缘或下三分之一 |
| 运镜 | 极缓慢平移、固定机位为主，禁用快速运镜 |
| 粒子 | 必须有（雾、尘、雪粒），但极薄 |
| 表演 | 内敛，几乎无表情，眼神戏为主 |
| 色温 | 完全偏冷，色温 4000K 以下 |

### Prompt 关键词

```
Roger Deakins cinematography, Denis Villeneuve aesthetic,
anamorphic lens, single hard light, deep shadows,
cold teal palette, rust red accents, negative space composition,
35mm film grain, slow camera movement
```

### 负面提示词

```
bright saturated colors, warm orange lighting, soft glow,
multiple light sources, flat lighting, cheerful mood,
quick camera movement, decorative elements
```

### 适合做什么

- 暗黑题材（修仙、悬疑、犯罪、科幻）
- 个人作品集（最易出"电影感"）
- 高级感商业片（但不适合活泼产品）

### 不适合

- 美妆 / 食品 / 母婴等需要"温暖"的题材
- 短视频信息流（节奏太慢）

## 沙丘古黑

**别称：** Greig Fraser 史诗系
**代表作：** 《Dune》《Dune Part Two》《The Batman》

### 核心特征

比蒙考拉更"古"、更"宗教感"。色调偏暖灰、土黄、墨黑，光源更戏剧化（顶光、剪影、烛光感），有古代典籍的厚重感。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 戏剧性顶光、剪影逆光，烛光感 |
| 色板 | 土黄、墨黑、暗红、铜锈绿，受难感 |
| 构图 | 中心对称、宏大尺度、人物极小 |
| 运镜 | 缓慢推拉，史诗感升降 |
| 颗粒 | 重颗粒，仿古胶片 |
| 表演 | 神圣感、肃穆，仪式化 |

### Prompt 关键词

```
Greig Fraser cinematography, Dune aesthetic,
biblical lighting, top-down god rays, dust particles,
desaturated earth tones, ochre and black, monumental scale,
ritual atmosphere, ancient texture
```

### 适合做什么

- 史诗题材（神话、宗教、历史）
- 文明陨落、时间永恒等宏大母题
- 高端品牌叙事

### 不适合

- 现代都市题材
- 轻快活泼的内容

## 栖地东方幻

**别称：** 东亚水墨电影美学
**代表作：** 《刺客聂隐娘》《一代宗师》《卧虎藏龙》

### 核心特征

水墨画质感的电影化呈现。雾气弥漫、留白大量、色彩极克制（偏青绿、墨黑、米白），动作戏反而以静制动。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 散射光为主，自然光感 |
| 色板 | 青绿、墨黑、米白、极少量朱红 |
| 构图 | 大量留白，主体偏角，模仿宋画 |
| 运镜 | 极慢、固定机位、长镜头 |
| 雾气 | 必备，但要薄、要灵动 |
| 表演 | 内敛克制，动作时极致放慢 |

### Prompt 关键词

```
Chinese ink wash painting cinematography, Hou Hsiao-Hsien style,
Wong Kar-Wai Ashes of Time aesthetic, ethereal mist,
ink black and bamboo green, traditional Chinese composition,
Song Dynasty landscape painting reference, negative space
```

### 适合做什么

- 古装、仙侠（避免脸谱化）
- 茶道、香道、东方哲学题材
- 中式高端品牌

### 不适合

- 紧凑叙事的现代题材
- 需要明快节奏的内容

## 辛麦贩赛博

**别称：** 赛博朋克霓虹系
**代表作：** 《Blade Runner 2049》《Cyberpunk 2077》《Ghost in the Shell》

### 核心特征

未来都市的雨夜，霓虹灯反射在湿漉漉的街道上。色彩高度饱和但只有特定几种（粉红、青蓝、紫罗兰），雾气、烟、雨是必备。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 多霓虹光源，色彩交错 |
| 色板 | 霓虹粉、电青、紫罗兰、深黑底色 |
| 构图 | 高角度俯视 / 低角度仰望，建筑感强 |
| 运镜 | 推进 + 横移，未来感运镜 |
| 雨雾 | 必备，营造氛围 |
| 反光 | 地面湿漉漉的霓虹倒影 |

### Prompt 关键词

```
cyberpunk aesthetic, neon-noir, rain-soaked streets,
blade runner 2049 atmosphere, pink and cyan neon,
volumetric fog, holographic projections, dystopian future,
wet reflective surfaces, anamorphic lens flares
```

### 适合做什么

- 科幻、未来题材
- 游戏宣传片
- 科技产品（前提是产品调性匹配）

### 不适合

- 传统、自然、温暖题材

## 王家卫潮湿

**别称：** 港片黄金时代美学
**代表作：** 《重庆森林》《花样年华》《一代宗师》

### 核心特征

手持镜头的不稳定感 + 慢动作 + 偏色的暖调（黄、红、绿） + 雨夜霓虹。情绪是暧昧、潮湿、孤独的都市感。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 现实光源（霓虹、路灯、台灯），偏色 |
| 色板 | 暖黄、酒红、墨绿，反差对比 |
| 构图 | 不工整、压抑、人物常被边框切割 |
| 运镜 | 手持微抖、慢动作切快动作 |
| 时间 | 静止、加速、慢动作切换 |
| 表演 | 凝视、独白、留白 |

### 适合做什么

- 都市情感
- 文艺向 MV
- 怀旧主题

### 不适合

- 需要客观叙事的题材

## 北野武极简

**别称：** 日式极简暴力美学
**代表作：** 《花火》《座头市》《菊次郎的夏天》

### 核心特征

极致克制 + 突然爆发。大量固定机位 + 空白构图，然后一个暴力或诗意的瞬间砸进画面。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 自然光为主 |
| 色板 | 蓝白主调，偶尔大面积红 |
| 构图 | 严格固定机位，对称或极端不对称 |
| 运镜 | 几乎不动 |
| 表演 | 面无表情，突然行动 |
| 节奏 | 静 — 静 — 静 — 爆 |

### 适合做什么

- 高级品牌（突出克制的力量）
- 艺术片
- 短篇叙事

### 不适合

- 信息密集的商业内容

## 韦斯·安德森对称

**别称：** Wes Anderson 童话美学
**代表作：** 《布达佩斯大饭店》《月升王国》《法兰西特派》

### 核心特征

完美的中心对称构图 + 平面化色彩 + 糖果色调 + 仪式化运镜。每一帧都像精心摆放的玩具屋。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 构图 | 严格中心对称，所有元素正面朝镜头 |
| 色板 | 糖果色，但每个场景一个主色 |
| 运镜 | 平移、推拉，绝不倾斜 |
| 表演 | 木讷感、仪式化 |
| 字体 | 衬线复古字体作为画面元素 |

### 适合做什么

- 复古品牌（书店、咖啡馆、文创）
- 童话题材
- 幽默向叙事

### 不适合

- 严肃 / 暗黑题材

## 塔可夫斯基长镜

**别称：** Andrei Tarkovsky 神性长镜头
**代表作：** 《潜行者》《镜子》《乡愁》《牺牲》

### 核心特征

极致缓慢的长镜头 + 潮湿环境 + 自然元素（水、火、土）+ 神性氛围。时间被拉到几乎停滞。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 自然光，雾天散射光 |
| 色板 | 棕褐、青灰、铁锈 |
| 构图 | 宽幅自然景观 |
| 运镜 | 极慢推拉，单镜头超过 1 分钟 |
| 自然元素 | 水滴、火焰、潮湿地面必备 |
| 时间感 | 几乎停滞 |

### 适合做什么

- 哲学题材
- 自然/宗教/信仰
- 作者电影

### 不适合

- 商业片（节奏太慢）

## 大卫·林奇梦魇

**别称：** David Lynch 超现实
**代表作：** 《穆赫兰道》《双峰》《橡皮头》

### 核心特征

色彩极端反差（红 vs 蓝），高噪点画质，慢动作与定格交替，配上不和谐音效。营造梦境般的不安。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 色板 | 极端的红+蓝对比，黑暗中的霓虹 |
| 噪点 | 重噪点，模拟梦境 |
| 运镜 | 不规则、有时颠倒 |
| 表演 | 木讷、不正常、过度反应 |
| 物体 | 日常物体被特写到失去意义 |

### 适合做什么

- 悬疑、心理题材
- 实验性艺术片
- 不寻常的概念片

### 不适合

- 任何商业用途

## 新海诚通透

**别称：** 新海诚动画系
**代表作：** 《你的名字》《天气之子》《铃芽之旅》

### 核心特征

极度饱和的蓝天 + 镜头光斑 + 透明感 + 青春期情绪。日漫平面美学的电影化。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 色板 | 饱和蓝、橙红夕阳、白云 |
| 光源 | 强烈光斑、丁达尔效应 |
| 透明感 | 玻璃、水、眼泪等透明物体 |
| 表演 | 青春化、敏感、内心戏 |
| 自然 | 云、雨、雪、樱花 |

### 适合做什么

- 校园题材
- 青春系产品（饮料、护肤、文创）
- 旅行 / 治愈系内容

### 不适合

- 严肃 / 暗黑题材

## 文艺复兴油画

**别称：** Caravaggio / Vermeer 古典美学
**代表作：** 《Barry Lyndon》（库布里克）《青年艺术家肖像》

### 核心特征

烛光级别的暗调 + 戏剧性单光源 + 重重阴影 + 古典构图。像把油画放进运动镜头。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 单一烛光级别光源 |
| 色板 | 深棕、暗红、土黄、墨黑 |
| 构图 | 古典三角形 / 黄金分割 |
| 运镜 | 几乎不动 |
| 服装 | 重纺织感 |
| 表演 | 仪式化、肃穆 |

### 适合做什么

- 历史 / 宗教题材
- 高端奢侈品
- 古典艺术内容

### 不适合

- 现代 / 科技题材

## 东亚日常物哀

**别称：** 是枝裕和系 / 小津安二郎系
**代表作：** 《步履不停》《海街日记》《东京物语》

### 核心特征

自然光 + 日常生活场景 + 极轻的悲伤。不戏剧化，不煽情，但每一帧都带着"时间流逝"的物哀感。

### 必守铁律

| 维度 | 规则 |
|------|------|
| 光源 | 完全自然光 |
| 色板 | 米白、浅灰、淡蓝、米黄 |
| 构图 | 平视，"榻榻米视角" |
| 运镜 | 固定机位为主 |
| 表演 | 日常化、不演技化 |
| 物件 | 茶杯、窗帘、植物、家具 |

### 适合做什么

- 家庭题材
- 日式品牌
- 生活方式内容

### 不适合

- 高对比戏剧化题材

## 风格融合警告

**不要试图融合多个风格。**

新手最常见的错误："我想要蒙考拉冷玩的色调，加上新海诚的光斑，再加点王家卫的手持感。"

这等于没有风格——最终出来的画面是各种元素的堆砌，没有任何一个美学是完整的。

**正确做法：选一个风格，从一而终，把那一个风格做到位。**

唯一例外：刻意的"风格冲突"作为叙事手段（如前半段栖地东方幻、后半段蒙考拉冷玩，象征角色心境转变）。但这需要明确的叙事理由，不是装饰。
